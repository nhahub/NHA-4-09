# 📦 Moodly App  

A modern **Flutter mental wellness app** designed to improve users' emotional and physical well-being.  
Built with **Clean Architecture**, focusing on performance, scalability, and smooth user experience.  

---

## 📖 About  

Moodly is a comprehensive **mental health & lifestyle application** that يساعد المستخدمين يفهموا مشاعرهم ويحسنوا نمط حياتهم.  
The app combines mood tracking, guided meditation, personalized recommendations, and therapist interaction — all in one place.  

---

## 🚀 Features  

### 🔐 Authentication  
- Sign-in / Sign-up  
- Secure authentication  
- Logout  

### 🚀 Onboarding & Personalization  
- Onboarding screens for smooth introduction  
- Questionnaire to understand user mood & preferences  
- Personalized recommendations based on answers  

### 🏠 Home & Daily Tracking  
- Dynamic home screen based on user mood  
- Water tracking system  
- Daily wellness insights  

### 🥗 Food & Recommendations  
- Recommended food based on mood  
- Detailed food screens (benefits, description)  

### 😊 Mood & Wellness  
- Mood-based recommendations  
- Meditation suggestions  
- Relaxation techniques  

### 🧘 Meditation & Audio  
- Guided meditation sessions  
- Audio player for relaxing content  

### 👨‍⚕️ Therapist Interaction  
- Therapist listing  
- Therapist details & reviews  
- Booking sessions بسهولة  
- Live session support  

### 💬 Communication  
- Real-time chat with therapists  
- Chatbot for instant help  
- Community screen for sharing experiences  

### 💎 Premium Features  
- Premium subscription access  
- Exclusive content & sessions  

### 💳 Payments  
- Secure payment screens  
- Subscription management  

### ⚙️ Settings & Profile  
- User profile management  
- App settings customization  
- Logout functionality
